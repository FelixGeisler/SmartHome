= 7. Deployment View

[plantuml,deployment,svg]
----
@startuml
node "Browser" {
  artifact "dashboard SPA" as spa
}
node "Host" {
  artifact "smarthome.jar" as app
  database "H2\n(embedded, in-memory)" as h2
  app --> h2
}
node "Smart device" as dev
spa --> app : HTTP, same origin\n(static assets + /api)
app --> dev : HTTP (LAN)
@enduml
----

[cols="1,3",options="header"]
|===
| Aspect | Current state
| Packaging | Single executable Spring Boot JAR; the Maven build compiles the SPA and bundles it under `classpath:/static`, so one artifact ships both
| Frontend delivery | Served by the JAR at `/`, same origin as the API — no CORS; the Vite dev server proxies `/api` during development only
| Datastore | Embedded in-memory H2 — not retained across restarts
| Device access | HTTP over the LAN, addressed by host
|===
