= 11. Risks and Technical Debt

[cols="1,3",options="header"]
|===
| Item | Description
| In-memory persistence | H2 runs in-memory, so registered devices are lost on restart. Cheap to pay down: switching to a persistent store (file-based H2 or Postgres) is mostly configuration — the Spring Data JPA code is unaffected.
| Unauthenticated API & unchecked device addresses | Any network client can register and control devices, and `externalId` is concatenated into outbound URLs unchecked, so a crafted value can aim the hub at an arbitrary host. Both fall under the Security module (M6).
|===
